# CSP Visual Projects

This repo contains a few CSP-based problems implemented in a way that’s easy to understand and visually intuitive, not just code-heavy.

## What’s inside

* **Map Coloring**

  * Australia (classic)
  * Telangana districts (scaled version)
  * Focus on constraints + clean visual output

* **Sudoku Solver**

  * Solves puzzles using CSP + backtracking
  * Uses MRV for better performance
  * Step-by-step solving visualization

* **Cryptarithmetic (SEND + MORE = MONEY)**

  * Assigns digits to letters under constraints
  * Ensures valid arithmetic solution
  * Includes simple tracing of steps

## Highlights

* Clean Python implementations
* Focus on clarity over complexity
* Visual outputs for better understanding
* Works smoothly in Jupyter

## Note

The idea here was not just to solve these problems, but to make the process easy to follow and visually clear.
